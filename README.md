# TU1 🐊

> *A sustainable token ecosystem with riddle-based mint, Uniswap V4 Hook dynamic fees, and self-sustaining agentic treasury.*

TU1 is a community-driven crypto project on **Base (Ethereum L2)**. It combines a fair-launch mint mechanism, automated fee distribution via **Uniswap V4 Hook**, and an AI-agent-managed treasury to create a token that sustains itself.

## Core Philosophy

| Principle | Implementation |
|-----------|---------------|
| **Fair Start** | Riddle-based mint — everyone plays the same game |
| **Anti-Bot** | Max 10 mints per wallet |
| **Sustainable** | Every mint fee feeds LP + Treasury, not just founders |
| **Self-Sustaining** | Treasury managed by AI agent for buyback, marketing, growth |

## Quick Stats

- **Supply:** 1,000,000,000 TU1
- **Network:** Base (Ethereum L2)
- **Mint Price:** $1 per 100,000 TU1
- **Max Mint:** 10 per wallet
- **Total Mint Supply:** 550,000,000 TU1 (55%)
- **Launch:** via Bankr → Uniswap V4
